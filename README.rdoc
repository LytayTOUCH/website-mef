== កំណត់ត្រាបច្ចេកទេសសម្រាប់បង្កើតគេហទំព័រ <b>អគ្គនាយកដ្ឋានលទ្ធកម្មសាធារណៈ</b>

* 1) <b>សកម្មភាពទី‌មួយ៖</b> សិក្សារពីតម្រូវការក្នុងការប្រើប្រាស់គេហទំព័រថ្មីរបស់លទ្ធកម្មសាធារណៈ

* 2) <b>តម្រូវការក្នុងការបង្កើតគេហទំព័រ៖</b>
	- ឈ្មោះរបស់គេហទំព័រលទ្ធកម្មសាធារណៈ (Domain Name) www.gdpp.gov.kh
	- ជ្រើសរើសសេវាកម្មក្នុងការបង្ហោះគេហទំព័រថ្មី (Hosting)
	- ជ្រើសរើសទីប្រឹក្សាក្នុងស្រុកសំរាប់ការបង្កើតគេហទំព័រ
